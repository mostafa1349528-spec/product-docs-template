# Product Documentation Template

This repository is a reusable template for documenting early-stage products,
MVPs, and startup ideas.

## How to use
1. Click **Use this template**
2. Name your new repository
3. Start filling documents from `01-vision`

## Folder Structure
- `01-vision` → Product vision, problem, personas
- `02-mvp` → MVP scope, success metrics
- `03-business-model` → BMC, pricing hypotheses
- `04-product-design` → Epics, user stories
- `05-validation` → Assumptions & experiments

## Versioning
Use versioned documents (v0.1, v0.2, …) to track learning over time.
